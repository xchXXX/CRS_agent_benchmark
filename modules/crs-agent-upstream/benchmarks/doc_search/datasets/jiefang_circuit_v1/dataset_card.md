# Jiefang Circuit v1

## 用途

该数据集用于评估 CRS-DocSearch-Bench 在**解放系列电路图/线束图**搜索场景上的召回与排序能力。

## 数据来源

- 来源：`/Users/zhangjiexiang/Desktop/doc_search_benchmark/资料树节点及关联文件(添加52份解放相关电路图搜索关键词语).xls`
- 原始行数：52 行（仅包含有搜索关键字的行）
- 生成 case 数：678

## 范围

- 主任务：`list_retrieval`
- 车型覆盖：解放 J5、J5P 奥威、J6、J6P（含牵引车、载货车、自卸车、天然气、领航版等细分）
- 资料类型：整车电路图、整车线束图、针脚定义、气路图、分册维修图等
- 输入：纯文字搜索关键词（无图片）
- Gold：每行对应的目标文件名称
- 当前不评估：澄清、页码、章节、文件内部内容片段

## Case 结构

每行 Excel 数据按 `搜索可能关键字` 拆分（中英文逗号分隔），每个关键词生成一个独立 case：

```
行数据：文件=A，关键词=B,C → 生成 case_AB, case_AC
```

所有 case 的 `answerable=true`，`image_paths=[]`。

## 注意

- 同一关键词可能出现在不同行（对应不同文件），各自独立成 case
- 该数据集来自资料树节点的关键词标注，仅覆盖解放系列电路图资料
