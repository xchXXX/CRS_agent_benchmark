# 聊天记录导出脚本与产物映射

## 目标

为真实聊天记录审阅提供独立导出能力，不影响 benchmark 运行主链路。

## 代码路径

- 导出脚本：`C:\projects\CRS_agent\benchmark\doc_search_bench\chat_export\export_wecom_chat_html.py`

## 逻辑拆分

1. `parse_args`
   - 解析 MySQL 连接参数、分页参数、导出目录参数

2. `fetch_room_summaries`
   - 以 `room_id` 为粒度聚合群聊
   - 输出消息数、起止时间

3. `fetch_room_messages`
   - 拉取单个群的完整消息字段
   - 保留 benchmark 后续回查所需字段

4. `display_message_text`
   - 按 `msgtype` 归一化展示文案
   - 文件、小程序、图片、撤回等类型都保留可读摘要

5. `render_room_page`
   - 生成接近微信聊天记录风格的静态页面
   - 服务侧消息右对齐，成员消息左对齐
   - 按日期插入分隔条

6. `render_index`
   - 生成导出批次索引页
   - 提供 room 级入口、消息量、时间范围、分页数

7. `render_manifest`
   - 生成导出批次元数据
   - 记录总群数、总消息数、消息类型分布、分页大小

## 产物映射

- `index.html`
  - 人工审阅入口

- `manifest.json`
  - 导出批次元数据

- `raw/room_summaries.json`
  - 群聊级统计摘要

- `rooms/<room_slug>/messages.json`
  - 单群原始消息证据

- `rooms/<room_slug>/page-*.html`
  - 单群分页后的可视化页面
